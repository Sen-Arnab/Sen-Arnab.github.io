### Step 1: Create a GitHub Repository

1. Go to [GitHub](https://github.com) and sign in
2. Click the **+** icon in the top right, select **New repository**
3. Name it: `yourusername.github.io` (replace `yourusername` with your actual GitHub username)
4. Make it **Public**
5. Click **Create repository**

### Step 2: Push Your Code to GitHub

Open your terminal in this directory and run:

```bash
# Initialize git repository
git init

# Add all files
git add .

# Commit your files
git commit -m "Initial commit - personal portfolio website"

# Add your GitHub repository as remote (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_USERNAME.github.io.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** tab
3. Click **Pages** in the left sidebar
4. Under **Source**, select **main** branch
5. Click **Save**

### Step 4: Visit Your Website

Your site will be live at: `https://yourusername.github.io`

It may take a few minutes for the site to go live.

## ğŸ“Š Adding Analytics (Track Your Visitors)

The website includes Google Analytics 4 code. To activate it:

### Google Analytics 4 (Free & Recommended)

1. Go to [Google Analytics](https://analytics.google.com/)
2. Sign in with your Google account
3. Click "Start measuring" or "Admin" (gear icon)
4. Create a new property:
   - Property name: "Arnab Sen Portfolio"
   - Select your timezone and currency
5. Click "Create" and accept terms
6. Choose "Web" platform
7. Enter your website URL: `https://yourusername.github.io`
8. Copy your **Measurement ID** (looks like `G-XXXXXXXXXX`)
9. Open `index.html` and replace **both instances** of `G-XXXXXXXXXX` with your actual Measurement ID
10. Push changes to GitHub

**What you'll see:**
- Real-time visitors
- Page views and sessions
- Geographic location of visitors
- Traffic sources (where visitors come from)
- Device types (mobile, desktop, tablet)
